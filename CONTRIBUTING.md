## Development

### Requirements

- [Node.js](https://nodejs.org)
- [pnpm](https://pnpm.io)

### Setup

Install packages:

```sh
pnpm i
```

### Developing

Start development server:

```sh
pnpm run dev
```

Run tests:

```sh
pnpm test
```

Format everything:

```sh
pnpm run format
```
